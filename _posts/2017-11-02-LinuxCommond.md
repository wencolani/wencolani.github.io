`cat file1`

`cat file1 | less`

`cat > file1`: use `Ctrl + d` to end input

`cat >> file1`

`cat file1 file2 file3`

`cat file1 file2 file3 > file4`

`cat file1 file2 file3 | sort > file4`

`cat - file1 > file2`: first copy file1 to file2, then add input something to file2
